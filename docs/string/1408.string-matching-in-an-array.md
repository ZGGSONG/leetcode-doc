## 解题思路

1. (self) 暴力解法，如果有包含的则直接添加 --内存消耗：2.2 MB, 在所有 Go 提交中击败了51.61%的用户

```go
var ret []string
for k1, v1 := range words {
	for k2, v2 := range words {
		if k1 != k2 && strings.Contains(v2, v1) {
			ret = append(ret, v1)
			break
		}
	}
}
return ret
```
